# Requirements: Lineaf Price Tracker

**Defined:** 2026-03-14
**Core Value:** Команда Lineaf в любой момент видит актуальные цены конкурентов и их динамику для принятия решений по ценообразованию.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Scraping

- [ ] **SCRP-01**: Парсер собирает каталог матрасов 160×200 с askona.ru (все страницы пагинации)
- [ ] **SCRP-02**: Парсер собирает каталог матрасов 160×200 с ormatek.ru (все страницы пагинации)
- [ ] **SCRP-03**: Парсер собирает каталог матрасов 160×200 с sonum.ru (все страницы пагинации)
- [ ] **SCRP-04**: Для каждого матраса собираются поля: модель, цена до скидки, цена после скидки, жёсткость, высота, наполнитель, материал чехла, вес на спальное место
- [ ] **SCRP-05**: Парсер корректно обрабатывает JS-рендеренные страницы через Playwright
- [ ] **SCRP-06**: Парсер обнаруживает новые позиции (товары, которых не было в прошлом запуске)
- [ ] **SCRP-07**: Парсер обнаруживает удалённые позиции (товары, которые были, но исчезли из каталога)

### Storage

- [ ] **STOR-01**: БД хранит каталог продуктов с дедупликацией (таблица products)
- [ ] **STOR-02**: БД хранит историю снапшотов цен с привязкой к дате сбора (таблица price_snapshots)
- [ ] **STOR-03**: БД хранит лог запусков парсера с результатами (таблица scrape_runs)
- [ ] **STOR-04**: Схема БД поддерживает отслеживание статуса активности товара (is_active)

### Dashboard

- [ ] **DASH-01**: Дашборд показывает таблицу текущих цен по конкурентам с фильтрацией
- [ ] **DASH-02**: Дашборд показывает графики динамики цен по выбранным товарам/конкурентам
- [ ] **DASH-03**: Дашборд позволяет экспортировать данные в Excel/CSV
- [ ] **DASH-04**: Дашборд показывает ценовой индекс (средняя цена категории по конкурентам)
- [ ] **DASH-05**: Дашборд показывает индикатор свежести данных (когда последний раз собирались)
- [ ] **DASH-06**: Дашборд показывает список новых и удалённых позиций

### Scheduling

- [ ] **SCHD-01**: Парсер запускается автоматически раз в неделю по расписанию
- [ ] **SCHD-02**: Лог успешных/неуспешных запусков доступен в дашборде

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Notifications

- **NOTF-01**: Уведомления в Telegram при значимых изменениях цен
- **NOTF-02**: Уведомления при появлении новых позиций у конкурентов
- **NOTF-03**: Еженедельный отчёт-дайджест

### Advanced Scraping

- **ASCR-01**: Прокси-ротация для обхода блокировок
- **ASCR-02**: Автоматическое обнаружение изменений структуры сайта
- **ASCR-03**: Парсинг дополнительных размеров матрасов

### Advanced Analytics

- **ANAL-01**: Сравнение характеристик между конкурентами (сопоставление аналогов)
- **ANAL-02**: Ценовая карта позиционирования (scatter plot цена vs характеристики)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Мобильное приложение | Веб-дашборд достаточен для команды |
| Автоматическое ценообразование (repricing) | Слишком сложно для v1, решения по ценам принимаются вручную |
| Многопользовательская авторизация | Внутренний инструмент для небольшой команды |
| Парсинг маркетплейсов (Wildberries, Ozon) | Другая структура данных, отложено |
| Парсинг конкурентов кроме Askona/Ormatek/Sonum | Три основных конкурента достаточно для v1 |
