# Lineaf Price Tracker

## What This Is

Сервис мониторинга цен конкурентов для бренда матрасов Lineaf. Автоматически собирает цены, характеристики и ассортимент матрасов 160×200 у трёх конкурентов (Askona, Ormatek, Sonum), хранит историю изменений и предоставляет дашборд для анализа ценовой динамики и позиционирования на рынке. Инструмент для команды Lineaf — помогает принимать решения по ценообразованию и отслеживать рыночные тренды.

## Core Value

Команда Lineaf в любой момент видит актуальные цены конкурентов и их динамику, чтобы принимать обоснованные решения по ценообразованию.

## Context

- Lineaf — бренд матрасов, конкурирующий с Askona, Ormatek и Sonum
- Ранее данные собирались вручную или полуавтоматическим парсером (пример: askona_160x200_full2705.xlsx)
- Excel-файл содержит образец структуры данных, которую нужно воспроизвести
- Дашборд будет использовать команда из нескольких человек
- Сначала локальная разработка, затем деплой на виртуальную машину

## Constraints

- **Tech stack**: Python (Scrapy/Playwright) + FastAPI + PostgreSQL — предпочтение пользователя
- **Deployment**: Сначала локально, затем перенос на VM
- **Scope**: Только матрасы 160×200
- **Sites**: askona.ru, ormatek.ru, sonum.ru
- **Frequency**: Еженедельный сбор данных
