  <files>src/lineaf/scrapers/askona.py</files>
  <action>
    Create `src/lineaf/scrapers/askona.py` with class `AskonaScraper(BaseScraper)`:

    **Constructor:**
    - `super().__init__(site_name="askona", catalog_url="https://www.askona.ru/matrasy/160x200/")`

    **`collect_product_urls(self, page) -> list[str]`:**
    - Iterate pages: `{self.catalog_url}?page={page_num}` starting at page 1.
    - On each page: extract `__NEXT_DATA__` JSON from page content using `re.search(r'<script id="__NEXT_DATA__" type="application/json">(.*?)</script>', content, re.DOTALL)`.
    - Navigate JSON: `data["props"]["pageProps"]["data"]["listing"]["items"]`.
    - Filter items where `item.get("type") == "p"` (product type, not banner/promo).
    - Extract URL: `"https://www.askona.ru" + item["data"]["productLink"]` — this includes the `SELECTED_HASH_SIZE` parameter for 160x200 pricing.
    - IMPORTANT: The `productLink` already contains `?SELECTED_HASH_SIZE=160x200-...` — use it as-is to get correct size-specific pricing.
    - Stop pagination when `items` list is empty or has no type="p" items.
    - Call `self.delay()` between page loads.
    - Return full list of absolute URLs.

    **`extract_product(self, page, url: str) -> dict`:**
    - Navigate to URL with `self.goto_with_retry(page, url)`.
    - Extract `__NEXT_DATA__` JSON (same regex pattern).
    - Navigate: `data["props"]["pageProps"]["productData"]`.
    - Extract characteristics: `pd.get("characteristics", [{}])[0].get("items", [])` → build dict of `{item["name"]: item["value"]}`.
    - Field mapping (Russian keys from RESEARCH.md):
      - `firmness` ← "Жесткость"
      - `height_cm` ← "Высота матраса"
      - `filler` ← "Наполнитель"
      - `cover_material` ← "Материал чехла" OR "Съемный чехол" (try both)
      - `weight_kg` ← "Вес на спальное место, кг" OR "Вес матраса" (try both)
    - Price extraction: `pd.get("price")` for sale price, `pd.get("oldPrice")` for original. These are integers in JSON — convert to Decimal via `Decimal(str(value))` if not None/0.
    - Return dict with keys: source_site="askona", source_url=url, name, firmness, height_cm, filler, cover_material, weight_kg, price_sale, price_original.
    - All optional fields default to None if not found. Do NOT log missing optional fields (per user decision).
    - Call `self.delay()` after extraction.

    **Error handling:**
    - If `__NEXT_DATA__` regex returns None on a page → log warning, skip that page/product.
    - If JSON path navigation fails (KeyError) → log warning with URL, return dict with only name and prices filled (characteristics will be None).

    **Register in run_scrapers.py:** Add `from lineaf.scrapers.askona import AskonaScraper` to the spider registry in run_scrapers.py (update the import and spider map).
